little ones is a sandbox soundboard system.
There is no point to this soundboard, it is an exercise of creating an interactive system using only javascript and as few libraries as possible.
The board starts off muted for your discretion. To unmute the board toggle the switch labeled 'mute'.

How to use:
- clicking on the board results in adding a node which will sound and be added to the current track
- the different colors represent notes in ascending order from left to right
- clicking on the numbers below the board will switch tracks
- the 'solo' toggle allows you to play one or all tracks upon clicking '▶'

Objects:
node - responsible for playing a sound associated with its position and image
track - holds an ordered list of nodes
